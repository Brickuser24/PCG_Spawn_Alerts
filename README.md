# PCG_Spawn_Alerts
Email Alerts for PCG Spawns

To use this, you'll have to upload main.py and Last_Spawn.txt and run them on a server of your choice
I use pylex nodes for this

You will have to enter your email and the app password for your email in the fields in main.py
Also add which pokemon you want alerts for in the same file

Run the code on the server and you should get an email whenever a pokemon you added spawns

Set up email push notifications on your device to never miss a spawn you want
